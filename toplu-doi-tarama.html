ℹ️ Nasıl Çalışır ve Gizlilik Hakkında

Toplu DOI Tarayıcı, akademik yayınlarınızı listelemeyi kolaylaştıran yerel odaklı (local-first) bir araçtır:

  • API Kaynağı: DOI sorgulamaları doğrudan resmi ve açık akademik veri tabanı olan Crossref API (api.crossref.org) üzerinden yapılır.
  • Eşzamanlı Sorgu: Sistem, tarayıcınızın donmasını önlemek ve API sunucularına aşırı yük bindirmemek için sorguları arka arkaya 3'lü gruplar (concurrency limit) halinde gerçekleştirir.
  • Gizlilik Politikası: Tüm işlemler tarayıcınızda (istemci tarafında) çalışır. Girdiğiniz DOI listesi veya sorgulanan makaleler hiçbir şekilde üçüncü şahıslarla paylaşılmaz veya DrTR sunucularına gönderilmez.
  • Özetlerin Durumu: Özet önce Crossref'ten, bulunamazsa OpenAlex'ten (Elsevier, Springer vb. dahil), yine bulunamazsa PubMed'den çekilir. Tüm kaynaklarda kayıtlı özet yoksa alan boş kalır.
