RefDown — Tek Tıkla Atıf Üretici

Açtığınız herhangi bir web sayfasından — makale, kitap, gazete, PubMed/arXiv/DOI/ISBN linki — tek tıkla atıf üretir. Tüm formatlama tarayıcınızda yapılır; hesap yok, sunucuya veri gitmez. APA, Vancouver, AMA, MLA, Chicago, Harvard ve daha fazlasını destekler.

⬇️ ZIP olarak indir 📥 Kurulum Adımları GitHub Reposu

📚 Yaygın Atıf Stilleri

APA 7, Vancouver, AMA, MLA, Chicago Author-Date, Chicago Notes, Harvard, IEEE.

🔍 Otomatik ID Tanıma

DOI, PMID, arXiv ID, ISBN otomatik algılanır; Crossref / PubMed / OpenLibrary / arXiv API ile bilgiler tamamlanır.

🔒 Local-first

Veriler tarayıcınızda kalır. Üyelik, hesap veya bulut senkronizasyonu gerekmez.

📤 Çoklu Export

Panoya kopyala, BibTeX (.bib), RIS (.ris) veya düz metin olarak indirin.

📥 Kurulum

  1. Repoyu indirin: ZIP olarak indirin ve klasörü açın, veya git clone https://github.com/afstudy20-gif/refdown.git.
  2. Eklenti sayfasını açın: Adres çubuğuna chrome://extensions yazın ve Enter.
  3. Geliştirici modunu açın: Sağ üstteki Developer mode / Geliştirici modu anahtarını aktif edin.
  4. Eklentiyi yükleyin: Load unpacked / Paketlenmemiş öğe yükle düğmesine basın ve indirdiğiniz refdown klasörünü seçin.
  5. Araç çubuğuna sabitleyin: Tarayıcı sağ üstteki puzzle (🧩) simgesinden RefDown'u bulun ve raptiye simgesine basarak sabitleyin.
  6. Kullanmaya başlayın: Herhangi bir makale sayfasına gidin, RefDown simgesine tıklayın, stil seçin, Cite this page.

Not: RefDown geliştirme aşamasındadır ve henüz Chrome Web Store / Edge Add-ons üzerinden yayınlanmamıştır. Developer mode ile yükleme şu an için tek seçenektir. Tarayıcı her açılışta kısa bir "Geliştirici modu uzantısı kullanılıyor" uyarısı gösterebilir — bu normaldir.

🔒 Gizlilik

RefDown local-first tasarlanmıştır. Hiçbir kullanıcı verisi, atıf geçmişi veya analitik bilgi DrTR sunucularına gönderilmez. Eklenti yalnızca aşağıdaki dış servislere doğrudan istek gönderir (atıf metadatasını tamamlamak için):

  • api.crossref.org — DOI çözümlemesi
  • eutils.ncbi.nlm.nih.gov — PubMed kayıtları
  • export.arxiv.org — arXiv kayıtları
  • openlibrary.org — ISBN çözümlemesi

Bu istekler standart HTTP GET isteğidir; kişisel bilgi (e-posta, IP, geçmiş) iletilmez — yalnızca tarama yapılan DOI/PMID/ISBN/arXiv ID gönderilir.
